# CV-QKD Thesis Simulations

This repository contains numerical simulations developed for my Master's thesis on passive state preparation in Continuous-Variable Quantum Key Distribution (CV-QKD).

## Features
- Secret key rate optimization
- Holevo bound calculation using symplectic eigenvalues
- Excess noise analysis
- Distance-dependent performance evaluation

## Parameters Used
- Fiber loss: 0.2 dB/km
- Detector efficiency: 0.5
- Electronic noise: 0.1
- Reconciliation efficiency: 0.95

## Requirements
- Python 3.x
- NumPy
- Matplotlib
- SciPy
